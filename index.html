GABMA Tecnologias

Backend

Son aquellas tecnologias que se utilizan en el lado del Servidor, utilizadas para gestionar todas las peticiones de información que le llegan y para gestionar las bases de datos alojadas en los mismos. La información una vez tratada le es devuelta al dispositivo para que sea visualizada en el dispositivo a través de las tecnologías FrontEnd. Las tecnologías más utilizadas en el BackEnd son PHP, Java, .NET, Phyton, MySql, etc.

java

Java

Java es un lenguaje de programación orientado a objetos que puede realizar programas con la posibilidad de ejecutarse en cualquier contexto.

php

PHP

Lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML.

.net

.Net

.Net provee un extenso conjunto de soluciones generales para la administración y ejecución de programas escritos con la plataforma de Microsoft.

python

python

lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma.

MySql

MySql

Es un sistema de gestión de bases de datos relacional y está considerada como la base de datos de código abierto más popular del mundo. Usado po Wikipedia, Goolgle, Facebook, Twitter y Youtube

FrontEND

Son aquellas que se utilizan en el lado Cliente, las que se utilizan en los diferentes dispositivosque utilizamos para conectarnos con el servidor a través de internet. Normalmente estas tecnologías y lenguajes vienen implementados en los diferentes Navegadores que existen, aunque no todos las implementan al 100%. Las tecnologías más utilizadas en el FrontEnd son HTML, CSS, JavaScript, jQuery, Ajax, BootStrap, Angular, etc.
